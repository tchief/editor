## Editor 

## [PLAY HERE](https://tchief.github.io/editor/)

Use hotkeys when you play:
- 's', 'c' - add square/circle
- 'z', 'x' - clean all/selected
- 'q', 'w' - random color/size

Click and drag any node.
Play around with random colors and sizes.
